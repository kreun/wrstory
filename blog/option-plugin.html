상담 챗봇 - 옵션 플러그인 적용

2021-06-10 15:15

카카오톡에서 선택형 입력을 받을 수 있는 옵션플러그인을 챗봇에 적용했습니다.
설문이나 요청사항 등을 고객으로 부터 받을 때 쉽게 받을 수 있는 플러그인입니다.





이전 방법

  • 01 오픈빌더에서 사용자 입력 받기 (필수파라미터)
  • 02 스킬을 이용해서 구글시트에 기록

대화 형식으로 사용자의 입력을 받는 방식보다, 한번에 입력을 받는 형식으로 하려면
옵션플러그인을 사용해야 합니다.

채널 1:1 대화에서 사용하는 옵션플러그인이 있고,
챗봇에서 사용하는 옵션플러그인이 있습니다.



옵션플러그인준비하기

  • 01 비즈니스 채널
  • 02 옵션플러그인 사용 시나리오 준비 (사용자에게 입력 받을 내용)
  • 03 옵션플러그인 화면 json 데이터 구성
  • 04 옵션플러그인 데이터 및 완료 url 개발
  • 05 카카오톡에 옵션플러그인 사용 요청

옵션플러그인을 사용하기 위해서 비즈니스 채널 신청 및 사용자 프로필 사용 설정을 해야 합니다.



옵션 플러그인 방식

  • 01 옵션 플러그인 실행 버튼은 스킬로 만들어야 합니다.
    (빌더에 없는 버튼 옵션 이며, 카카오톡에서 전달하는 문서를 참고 합니다.)
  • 02 옵션 선택이 완료되고, 사용자에게 보여지는 화면은 이벤트가 설정된 블록입니다.
  • 03 옵션 데이터를 서버에서 전달 받고 구글 시트에 기록합니다.

저장데이터 확인
