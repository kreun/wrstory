오픈빌더 스킬 API로 확장하기

2020-11-28 15:25

개발자 없이, 코딩없이 스킬 개발 할 수 있다면 좋지 않을까요?


미리 만들어 둔 스킬 확장 API로 카카오톡 챗봇의 기능을 확장합니다.
스킬의 경우에는 코딩의 문제가 아니라, 서버 설정 부터 시작해야 하니
(서버리스의 경우에도 설정이 들어가긴 마찬가지 입니다.)
어렵고 힘든 과정입니다.
빌더에서 직접 입력해서 데이터가 저장되고,
저장된 데이터를 사용할 있고, 외부의 데이터도 사용할 수 있도록 합니다.


오픈빌더에서 설정만으로 미리 만들어둔 확장 기능을 사용할 있으니,
개발 부담 없이 만들고 싶은 챗봇을 만드세요.


Dialogflow 풀필먼트 확장 API도 지원합니다.





action : saveParameters
파라미터를 저장합니다.


속성값


pName : 저장할 파라미터 이름입니다. 여러인 경우에는 “,” 로 구분합니다.
pGroup : 파라미터 값 저장 위치
값이 없을 경우에는 신규 데이터로 업데이트 하며,
값이 있을 경우에는 해당 그룹에 데이터를 추가 한다.


사용 방법


  • 1 정해진 값을 저장하는 경우
    • 사용자의 입력값이 아닌, 특정 값을 저장하고자 할 때 (사용자 태그정의, 컨텍스트 전달 데이터 등)

  • 2 사용자 입력데이터를 저장하는 경우
    • 발화에서 인식된 파라미터값 (엔티티)
    • 되묻기 질문으로 받은 사용자 발화 자체


어디에 사용할 수 있을까요?


설문 챗봇을 만들고, 사용자 입력값을 저장할 때
퀴즈 챗봇을 만들고, 맞고 틀렸는지 저장할 때
사용자 정보를 받아서 저장할 때
